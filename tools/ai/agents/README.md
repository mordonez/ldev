# Agents Directory

Vendor-neutral home for multi-agent collaboration assets.

## Structure

- `context/`: manifests, generators, and coverage audit for LLM context packs.
- `roadmaps/`: planes de evolución y simplificación del pipeline de agentes.

## Canonical Rule

Use `agents/context/` como ubicación canónica para workflows de contexto compartido, y `agents/roadmaps/` para planes de reducción de deuda técnica.
